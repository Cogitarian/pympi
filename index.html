Installation¶

Run: # python setup.py install

If you don’t have right on your system to install in the default package location for python you can specify your own package location: $ python setup.py install --prefix=/home/user/bin

Note

You have to add export PYTHONPATH=~/home/user/bin/lib/python2.7/site-packages to your ~/.bashrc
